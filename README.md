# Generative art
## Description
The main objectives of this pet project are to practice using jQuery and learn more about generative art.
## How to use
### Requires:
* Node.js v.10.15.3
* Gulp v4
* npm last version
## Main commands
* Install - `npm i`
* Start local server - `npm start`
* Build without starting local server - `npm run-script build`
